Step 01

Clone and run.

No web host needed. No accounts. Clone the repo, install dependencies, and run the TUI locally. Your code never leaves your machine.

$ curl -fsSL https://raw.githubusercontent.com/Tlkh201313/awakening-cli-new-gen/dev/setup.sh | bash
# or Windows: irm .../setup.ps1 | iex
$ awakened
