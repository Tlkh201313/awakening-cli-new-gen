# VoltAgent / officialskills Productivity Guide

Upstream: https://github.com/VoltAgent/awesome-agent-skills — **1,100+** official-team curated skills.

## Browse & install

- Clone or follow install paths in the repo README (Claude Code, Cursor, Codex layouts differ).
- Browse via **officialskills.sh** when the user wants a catalog UI.

## How to work

1. Match intent to **one** skill per task — do not load multiple SKILL.md files.
2. **Read** only the relevant `SKILL.md` from the install path.
3. Do not paste the whole catalog into context.

## Distinction

| Pack | Source | Focus |
|------|--------|-------|
| **Awakened Skills Vault** | antigravity-awesome-skills | Community multi-domain playbooks |
| **Awakened Productivity** | VoltAgent awesome-agent-skills | Official-team curated integrations |

Use Vault for broad @-skill playbooks; use Productivity for Stripe, Vercel, Anthropic patterns, etc.
