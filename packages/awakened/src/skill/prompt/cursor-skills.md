# Cursor Skills Directory (skills.sh)

Browse: https://skills.sh/

## When to use

User wants Cursor/Codex/Claude Code skills from the open directory, `npx skills add`, or skills.sh packages.

## Install

```bash
npx skills add <owner/repo>
```

Browse https://skills.sh/ for React, Next.js, deploy, and framework skills.

## Rules

1. One SKILL.md per task — never paste whole catalogs.
2. Prefer built-in `frontend`, `testing`, `docs-writer` when they already cover the task.
3. After install, read the skill file before acting.
