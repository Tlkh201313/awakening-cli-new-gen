# Vercel Agent Skills

Upstream: https://github.com/vercel-labs/agent-skills  
Browse: https://officialskills.sh/

## When to use

Deploy to Vercel, Next.js App Router, React performance, Turborepo, Vercel platform APIs.

## Install

Clone vercel-labs/agent-skills or install via officialskills.sh / npx skills.

## Rules

1. Prefer `frontend` built-in skill for general UI; use Vercel skill for deploy/platform specifics.
2. One skill per deploy or framework concern.
3. Check project already on Vercel before suggesting new project setup.
