# Composio Skills

Upstream: https://github.com/ComposioHQ/awesome-claude-skills

## When to use

Integrations with SaaS (Slack, GitHub, Gmail, Linear, Stripe, Notion, …) via Composio toolkits.

## Install

Clone ComposioHQ/awesome-claude-skills and load **one** SKILL.md matching the integration.

## Rules

1. Match the exact service — do not load unrelated integration skills.
2. Verify Composio API keys / MCP config before calling tools.
3. Use awakened MCP + Composio docs for server setup when needed.
