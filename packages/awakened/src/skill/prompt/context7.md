# Context7

Fetch up-to-date library documentation instead of guessing APIs.

## Setup

Configure Context7 MCP in `awakened.json` with `CONTEXT7_API_KEY`.

## Workflow

1. Resolve the library ID in Context7 before answering version-specific questions.
2. Quote doc snippets — do not invent signatures.
3. Prefer Context7 over generic web search for framework APIs.

## When to use

"How do I use X in React 19?", Stripe API changes, Next.js App Router APIs, etc.
